# Healthy Res | Health Restaurant Menu 🥗

A modern, fast, and fully responsive landing page for a healthy food restaurant. Built using core web technologies and optimized for all devices, it delivers a seamless user experience from browsing nutritious meals to completing secure online checkouts.

## 🚀 Live Demo
Check out the live website here: [helthyres.netlify.app](https://helthyres.netlify.app/)

## ✨ Key Features
* **Dynamic Shopping Cart:** Add or remove items and watch the total order price update instantly without page reloads.
* **💳 Integrated Payment Gateway (Moyasar):** Seamlessly embedded with the official Moyasar Payment Form SDK[cite: 1].
* ** Pay & Credit Cards:** Fully supports quick checkouts via Apple Pay, Mada, Visa, and Mastercard[cite: 1].
* **Modern UI/UX:** Clean, elegant, and responsive layout styled cleanly with Tailwind CSS[cite: 1].

## 🛠️ Tech Stack
* **Structure & Styling:** HTML5 & [Tailwind CSS](https://tailwindcss.com)[cite: 1]
* **Cart Logic & Interactivity:** Pure JavaScript (Vanilla JS)[cite: 1]
* **Payment Gateway:** [Moyasar Web Form SDK](https://moyasar.com)[cite: 1]
* **Hosting Platform:** Netlify
